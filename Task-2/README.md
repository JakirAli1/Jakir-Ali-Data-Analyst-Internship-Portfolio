## Task 2: Exploratory Data Analysis (EDA)

**Objective:**  
To explore the dataset in depth and uncover patterns, trends, and insights that provide direction for further analysis and business decisions.

### Steps Performed:

1. **Data Overview & Summary Statistics**
- Reviewed summary statistics using `describe()`
- Checked distribution of key variables

2. **Univariate Analysis**
- Analyzed single variable distributions for key numeric fields
- Plotted histograms and value counts to understand data spread

3. **Bivariate Analysis**
- Explored relationships between pairs of features
- Used visualizations such as scatter plots and bar charts

4. **Trend Analysis Over Time**
- Converted date columns to datetime format
- Plotted trends over months/days for key metrics

5. **Category / Group Analysis**
- Analyzed group performance (e.g., sales by category)
- Compared trends across different segments

6. **Key Insights**
- Identified top performing categories
- Noted seasonal trends or peak activity periods
- Highlighted patterns that require deeper investigation

### Tools & Technologies Used:
Python, Pandas, Matplotlib, Seaborn, Plotly
