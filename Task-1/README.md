##  Task 1: Data Understanding & Cleaning

Objective:
To understand the structure and quality of the given dataset and prepare it for further analysis.

Steps Performed:

1. Data Load & Overview  
- Loaded the dataset using pandas  
- Checked dataset structure using head(), info(), and describe()

2. Missing Value Handling  
- Identified missing values  
- Filled numerical missing values using median  
- Filled categorical missing values using mode  

3. Duplicate Records Removal  
- Removed duplicate rows to ensure data consistency

4. Data Type Correction  
- Converted columns like dates and numerical fields to appropriate data types

5. Outlier Detection & Handling  
- Used boxplots to detect outliers  
- Removed or capped extreme outliers

6. Final Clean Dataset  
- Prepared a clean dataset ready for analysis  
- Saved the cleaned dataset for future tasks

Tools & Technologies Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
